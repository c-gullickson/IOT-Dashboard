IOT Dashboard Application Application Overview

This applicaiton is for the NDSU course CSCI 846 (Distributed Systems). It demonstrates the ability of distributed applications by integrating with IOT devices.

  • Ring Doorbell
  • Roku Television
  • Fitbit Watch
  • Smart Lights
  • Weather App

The purpose of this application is to manage tasks for a group of users. Tasks can be added to a backlog of total tasks, assigned, and completed. Once a task has been completed, it assignes a monetary value to the user whom completed the task. There is also a feature for the user to view that monetary amount, and perform purchases (withdrawls) against that value, overall keeping a running tab of events.
